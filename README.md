# FINANCE - Grupo AHAV

Aplicativo web interno para gestão do FINANCE do Grupo AHAV.

## Módulos

- Executivo
- Plano de Ação
- FVC - Valor Gerado
- Gestão de Riscos
- Análise de Crédito
- Smartmold ERP

## Publicação no Netlify

Este repositório está preparado para publicação pelo Netlify.

- Pasta publicada: `plano-acao-ahav-online`
- Arquivo principal: `plano-acao-ahav-online/index.html`

Ao conectar o repositório no Netlify, use as configurações do arquivo `netlify.toml`.

- `plano-acao-ahav-online/index.html`
